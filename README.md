# 🎬 Movie Explorer

Uma aplicação web moderna para explorar filmes e séries, desenvolvida com **React** e **TypeScript**, consumindo dados em tempo real da API do **The Movie Database (TMDB)**. O projeto permite navegar por diferentes categorias, visualizar detalhes dos títulos e assistir aos trailers disponíveis diretamente pelo YouTube.

---

## 📖 Sobre o Projeto

O **Movie Explorer** foi criado com o objetivo de praticar o consumo de APIs REST, gerenciamento de estado, componentização e desenvolvimento de interfaces modernas utilizando React.

Todas as informações são obtidas em tempo real através da API do TMDB, mantendo o catálogo sempre atualizado.

---

## ✨ Funcionalidades

* 🎥 Exibição de filmes em destaque.
* 🔥 Categorias de filmes.
* 🔍 Pesquisa de filmes.
* 📄 Página de detalhes do filme.
* ▶️ Reprodução de trailers disponíveis no YouTube.
* ⭐ Exibição de avaliação dos filmes.
* 🖼️ Posters e imagens oficiais.
* 📱 Interface responsiva.

## 🚀 Como Executar

### Clone o repositório

```bash
git clone https://github.com/mariayasmim2904-alt/App-de-Filmes.git
```

### Entre na pasta

```bash
cd aplicativo_de_filmes
```

### Instale as dependências

```bash
npm install
```

### Execute o projeto

```bash
npm run dev
```

---

## 🔑 Configuração da API

Crie um arquivo `.env` na raiz do projeto e adicione sua chave da API do TMDB:

```env
VITE_TMDB_API_KEY=SUA_CHAVE_AQUI
```
